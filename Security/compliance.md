
## TCSEC (橙皮书)

《可信计算机系统评价准则》(Trusted Computer System Evaluation Criteria，TCSEC)
由美国国家安全局(National Security Agency)和其他美国政府机构在20世纪80年代初发布

## CC

《信息技术安全评价通用准则》(Common Criteria(CC) for Information Technology and Security Evaluation)
是多国在20世纪90年代末开发的通用准则(ISO/IEC 15408)

## 等保

《网络安全等级保护》
是中国的国家标准(GB 17859系列)

### 国密

包含: SM2(非对称) / SM3(HASH) / SM4(对称)

如果任何人都可以自由实现 SM2 算法，如何确保实现不出错、没有后门？
这就需要商用密码产品的认证体系。
一个算法实现最终要被市场（尤其是政府、金融行业）认可，必须通过国家密码管理局的检测，并获得《商用密码产品认证证书》
