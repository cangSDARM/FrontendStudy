## 分类

### 恶意软件(Malware)

- 病毒(感染)：修改正常程序使得病毒进行复制而进行感染的软件
- 蠕虫(传播)：主动寻找并感染其他机器的程序
- 木马(窃取)：表面上看起来有用的程序，但内部藏有恶意代码

四大类有效载荷

1. 数据窃取(Data Exfiltration)
2. 远程访问(Remote Access)
3. 系统破坏/扰乱(System Disruption)
4. 自主传播(Propagation)

### 网络威胁

- DoS: 通过耗尽系统资源，来减少系统的可用性
  - 网络链路攻击:
    - 直接攻击: 攻击者使用 ICMP/UDP/TCP SYN/SIP 洪泛等直接攻击目标 ip
    - 反射攻击: 攻击者伪造源 ip 为目标 ip, 发送给中继，中继反射流量给目标 ip
    - 放大攻击: 反射攻击的中继会反射过大的流量给目标 ip
  - 防范:
    - 路由过滤器: 路由验证来源 ip 是否是有效/发送地址，防止伪造
    - syn cookie: 服务器回复带握手信息 cookie 的 syn，发送后不等待；客户端回复 ack 有有效 cookie 时再重建链接
    - 修改配置: 超时窗口、连接表
    - CDN/用户验证/关闭广播请求
- 劫持
  - 点击劫持
  - DNS 劫持

### 应用与代码威胁

- APT 高级持续性威胁
