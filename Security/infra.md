## 云基础设施

“云”即云计算，是通过网络，弹性、按需、付费使用 IT 资源(基础设施 IaaS、平台 PaaS、软件 SaaS)的服务

四大部署模式（架构落地形态）

- 公有云：第三方服务商提供，面向公众开放，按需付费
- 私有云：为单一组织专属搭建，安全可控，适合政企核心业务
- 混合云：公有云 + 私有云结合，兼顾弹性与安全，是企业主流选择
- 社区云：多个有共同需求的组织共享的云环境（如行业联盟云）

参考架构

|    角色     |                                  核心职责                                   |
| :---------: | :-------------------------------------------------------------------------: |
|  云消费者   |             按需使用云服务，向云提供商付费，是服务的最终使用者              |
| 云提供商CSP |    负责云基础设施的建设、运维、管理，向消费者交付云服务，是服务的供给方     |
|  云代理商   | 为消费者提供云服务选型、集成、管理、优化等中介服务，解决多云选型 / 管理问题 |
|  云审计者   |     独立评估云服务的安全性、合规性、性能，出具审计报告，保障云环境合规      |
|  云承运商   |      负责云服务的网络传输，保障消费者与云数据中心之间的网络连通与安全       |
